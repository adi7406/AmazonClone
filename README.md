
🛒 Amazon Clone (HTML & CSS)

A simple Amazon homepage clone built using HTML and CSS.
This project focuses on replicating the layout and design of Amazon's landing page to practice frontend development and responsive design concepts.

📌 Features

Amazon-style navigation bar

Search bar UI

Product grid layout

Hero banner section

Footer similar to Amazon

Clean and responsive CSS styling

Fully built using pure HTML and CSS

🛠️ Technologies Used

HTML5 – Page structure

CSS3 – Styling and layout

Flexbox & Grid – Layout management

🎯 Learning Objectives

This project helped practice:

Structuring web pages using HTML

Styling layouts with CSS

Using Flexbox and Grid

Building real-world UI clones
